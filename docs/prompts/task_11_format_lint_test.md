# Контекст
У нас образовалась проблема в линтинге, тестировании и форматировании. Нам нужно привести проект в порядок
# Backend
Основной backend и микросервисы имеют `Makefile`
В этом файле обязательно должны быть команды `lint`, `test` и `format`
# Frontend
Там есть аналогичный Makefile, но там не реализована команда format. Необходимо её реализовать
# Агенты
Профильные backend и frontend агенты обязаны перед сдачей работы на QualityGate обязаны выполнить эти три команды и убедиться, что всё проходит успешно
Сам QualityGate тоже обязан запустить эту команду в каждом затронутом микросервисе
# Форматирование
При форматировании очень важно не затронуть папки `.github` и `.helm`, которые либо сейчас есть, либо позже будут в каждом репозитории
# Задача
Необходимо доработать профильных агентов, QualityGate и привести основные сервисы и микросервисы в состояние отсутствия ошибок по всем трём командам
Не забудь на фронте реализовать `make format`

# Library
Нет необходимости реализовывать UI элементы самостоятельно с нуля. Давай воспользуемся готовой библиотекой MaterialUI
Сайт и документация: https://mui.com/material-ui/getting-started/installation/
Установи эту зависимость и перепиши страницы с использованием этой библиотеки
# Dark Theme
Все страницы должны поддерживать как светлую, так и тёмную тему оформления. Зависит от системных настроек ОС, с которого запущен сайт
# Login page
Страница регистрации и авторизации `/login`. 
Выглядит отлично. Не хватает только расстояния между Radio логина и регистрации аккаунта
Давай использовать [Toggle Button](https://mui.com/material-ui/react-toggle-button/) для переключенния между режимами
# Main Page
Основная страница на данный момент должна содержать функционал связи аккаунта с аккаунтом HeadHunter. Было требование реализовать следующие действия:
- привязка нового аккаунта
- удаление аккаунта (не обязательно активного, но и любого из списка)
- переключение между аккаунтами
В рамках одного выпадающего меню. На данный момент все элементы при нажатии на кнопку тупо располагаются в шапке
Можно для этих целей использовать что-то типа [Drawer](https://mui.com/material-ui/react-drawer/), которых выходит сверху
В меню пользователя необходимо реализовать действие выхода из аккаунта
